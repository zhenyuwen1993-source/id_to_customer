## 二、设计概念

这套房的本质是「自住 + 包租」的**合居空间**——方案核心不只是好看，而是**温润的同时经得起合租**：隔音私密、收纳充裕、耐用易打理。

以**暖极简**为底：燕麦白 + 暖橡木 + 一处陶土点睛，营造让你们愿意共享、也让租客一眼想签约的**沉静有家感**公共区；三间卧室都做成**带锁、带衣柜、安静**的私人小天地。预算花在刀刃上——**厨房（玻璃门封闭+强排）、隔音、收纳、耐用地材、出租回报**优先，装饰其次。风水与佛龛不是事后加装，而是**从布局起就融入**。

**风格关键词**：`暖极简` `合租友好` `隔音私密` `收纳充裕` `耐用易打理` `自然温润`

## 四、主要材质

- **暖橡木纹 SPC/强化地板** —— 耐磨防水易清洁，合租首选，全屋连续显大
- **岩板 / 石英石台面 + 易清洁防溅墙** —— 重油烟厨房经得起造
- **厨房玻璃移门** —— 挡油烟气味，又通透明亮
- **亚麻 / 棉麻软装 + 厚窗帘 + 实芯门 + 门缝密封条** —— 自然吸音、卧室隔音
- **微水泥 / 浅石材**（佛龛背景）+ 暖白乳胶漆；藤编、木质点缀

## 五、分区与家具（逐空间）

- **玄关**：通顶鞋柜（按 4 人量）+ 换鞋凳 + 挂钩 + 感应灯；**陶土端景**（仪式感 + 玄关挡煞，入门不见灶厕）。
- **客厅（共用）**：耐脏科技布沙发、暖木电视柜（大收纳）、**大餐桌兼共享工作台**、吊扇 + 柔光；**佛龛端景墙 + 小水景**在此（见下）；南面临学校处厚帘 + 隔音玻璃。
- **餐厨（共用 · 玻璃移门封闭）**：强力抽油烟、岩板台面、**足量吊柜地柜（分区给各住户存物）**、双门冰箱位；灶位排好动线（见风水）。
- **主卧（业主 · 带主卫）**：定制衣柜、暖木背景墙、柔光、隔音门；床头靠实墙、不对门；主卫舒适升级。
- **次卧 ×2（出租）**：各配**定制衣柜 + 书桌 + 优质床垫 + 遮光帘 + 门锁 + 吸音处理**；中性暖白，好租耐看。
- **公卫（租客共用）**：耐用易清洁、干湿分离、充足挂置、防滑地面。

## 六、🧭 风水兼顾 + 小佛龛融入设计

### 小佛龛（神位）
- **位置**：共用客厅区（神位宜厅不宜卧）。靠一面**实心承重墙**做靠山，**朝大门方向**（迎气），前方明堂开阔、采光好。
- **规避**：背后不靠厕所/灶台/床头；**不压梁**（吊顶包梁或错位）；不正对厕所门/卧室门/灶；不被空调直吹；**抬升至恭敬高度**（神像视线高于人站立视线）。
- **融入暖极简**：**暖橡木悬浮龛 / 嵌入式壁龛** + 微水泥/浅石材背景，**3000K 隐藏暖光**洗墙；台下隐藏供品收纳，留**通风缝**。
- **合租友好**：加**小排风或电子供灯/无烟香**，减少香火对租客影响，更安全；保持整洁留白（恰是暖极简语言）。

### 其余破局（与设计同步落地）
- **九运火旺（南/学校面）**：客厅一处**小水景/鱼缸**平衡火气，兼装饰与聚气（呼应 Alkaff 湖）。
- **灶位**：不对门、不背靠厕所、不与水槽/冰箱正面相冲。
- **玄关**：入门不见灶厕 → 用陶土端景化解。
- **主卧**：床头靠实墙、不对门、不压梁、不对镜；主卫门不正冲床。
- **财位**（进门斜对角）：整洁明亮，放绿植/暖光，不堆杂物。

## 七、2D 布局要点（保持 HDB 4 房原结构，不动承重）

入户 → **玄关收纳 + 端景** → 共用**客餐厨**（L 型动线，厨房在尽端以玻璃门封闭）→ **睡眠区三间卧室成组**：
- **主卧 + 主卫**（业主），床头靠实墙；
- **次卧 ×2 + 公卫**（租客）；卧室间以衣柜/软包作隔音缓冲。
- **佛龛**置于客厅一面实心墙、朝大门；**小水景**邻近客厅。
